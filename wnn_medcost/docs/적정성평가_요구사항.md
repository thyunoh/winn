- **내용**: 전월 유치도뇨관 있는 대상자(제거 대상자 포함)를 당월 화면에서 체크박스로 확인 가능하도록
- **구현**: select_CategoryList05 SQL에 prevMonth 컬럼 추가 (EXISTS 전월 조회), 헤더에 "전월" 컬럼, ✔ 표시
- **관련파일**: assessment.jsp, Magam_SQL.xml, PatvalDTO.java

## [대기] 유치도뇨관 전월 vs 당월 오류 자동 감지
- **내용**: 전월에 있었는데 당월에 빠진 환자 경고, foley체크/14일초과 항목 간 불일치 오류 표시
- **구현방향**: 전월 대상자 중 당월 미존재 환자 조회 SQL + 화면에 경고 목록 표시
- **관련파일**: assessment.jsp, Magam_SQL.xml

## [대기-엑셀양식 수령 후 진행] 유치도뇨관 엑셀 업로드 오류 크로스체크 기능
- **내용**: EMR/OCS의 유치도뇨관 삽입 오더 내역을 엑셀로 업로드하면, 평가표 상 삽입일자/제거일자와 자동 대조하여 오류 검출
- **오류 케이스**:
  - 오더에 삽입 있는데 평가표에 누락
  - 오더 삽입일 ≠ 평가표 개시일 (날짜 불일치)
  - 오더에 제거 기록 있는데 평가표에 제거일 미입력(이내용은오류제외)
  - 평가표에 있는데 오더에 해당 삽입코드 없음
- **구현방향**: 엑셀 파싱 → 평가표 데이터 조회 → 환자별 매칭 비교 → 오류 목록 표시
- **관련파일**: assessment.jsp, Magam_SQL.xml

## [대기] 구간별 대상자 수 박스 표시 (병원검색 화면)
- **내용**: 5점구간 컬럼에 5점뿐 아니라 4점·3점·2점·1점 구간별 필요 인원을 박스로 표시
- **단위**: 구간 기준값·필요 인원 모두 `명`으로 표시 (% 아닌 분모 기준 환산)
- **표시 예시** (장기입원 분모165, 분자67):
  - 5점: ~32명 | -35명 필요
  - 4점: ~49명 | -18명 필요
  - 3점: ~65명 | ★현재 67명
- **구현방향**: 5점구간 셀 클릭 시 팝업/툴팁으로 구간별 현황 표시
- **관련파일**: simulation.jsp
- **관련화면**: 병원검색 > 기간별 보기

## [대기] simulation.jsp 헤더-그리드 컬럼 정렬 불일치
- **내용**: DataTables scrollY 사용 시 헤더와 데이터 컬럼 너비가 어긋남
- **원인**: scrollY 적용 시 헤더/바디가 별도 table로 분리되면서 너비 불일치
- **해결방향**: 데이터 로드 후 `dataTable.columns.adjust()` 호출
- **관련파일**: simulation.jsp
