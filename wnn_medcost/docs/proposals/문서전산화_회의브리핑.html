위너넷 컨설팅 · WinCheck

컨설팅 문서 전산화 — 회의용 정리

한마디로

종이·팩스로 주고받던 의뢰서·견적서·홍보물을, 병원이 화면에서 직접 작성·동의·제출하고 위너넷은 한 곳에서 관리하도록 프로그램(WinCheck) 안으로 옮깁니다.

1무엇을 바꾸나 — 문서 5가지

문서지금바뀌면순위
의뢰서 + 동의서인쇄→작성→팩스 회신→화면에서 입력 + 동의서 사인1순위
견적서병원마다 수기 작성→병원 고르고 금액만 → PDF 자동 발행1순위
홍보물·회사소개바뀔 때마다 재제작→화면에서 문구·숫자 즉시 수정2순위
프로그램 사용법별도 안내문→앱 안 도움말·첫화면 안내3순위
연간 일정 안내안내문 발송→공지 + 월별 일정표3순위

핵심은 1순위(의뢰서·견적서) — 실제 수기 업무(접수·견적)를 없애는 부분입니다. 나머지는 순차 확장.

3비용은?

서버·프로그램 추가비용 없음

이미 쓰고 있는 무료 오픈소스로 만듭니다. 새 장비·서비스 구매 불필요.

딱 하나 짚을 점 — PDF 만드는 도구

견적서 PDF를 만드는 도구(iText)가 상용 판매 시 라이선스 비용이 생길 수 있습니다. → 무료 도구(OpenPDF)로 바꾸면 비용 0. 바꾸는 부담도 적습니다. (문자 알림·인증 사인은 원할 때만 유료)

5동의서 사인 — 두 방식 쉬운 비교

간단 사인 추천인증 사인
방법화면에 손가락·마우스로 사인휴대폰 인증·공동인증서
본인 확인이름 입력 + 접속기록있음 (강력)
법적 효력동의 증거(시간·IP)강함
비용무료유료(외부 서비스 계약)
준비바로 가능외부 연동·계약 필요
언제대부분 컨설팅 동의엔 충분분쟁·감사 리스크 클 때

인증 사인이 필요하면 — 휴대폰 인증(카카오·PASS) 또는 전자계약 서비스(모두싸인 등)를 유료로 연동하면 됩니다. 자체 구축은 과하니, 필요할 때 서비스 연동으로. 대부분은 간단 사인 + 접속기록으로 충분합니다.

6동의서를 어떻게 받나 — 방문 vs 원격

진짜 갈림길은 기기(폰·태블릿)가 아니라 방문이냐 원격이냐입니다. 이에 따라 서명 방법이 갈립니다.

방식서명 상황서명 방법
방문위너넷이 태블릿 지참 → 현장에서 대표가 손가락 사인화면 사인 (깔끔)
원격병원에 링크·QR 전달 → 병원이 자기 기기로 사인폰·태블릿=화면 사인 / 데스크탑=타이핑 사인

관건 = 원격 + 데스크탑 병원

위너넷은 원격 위주이고, 병원 접수 담당자는 대부분 터치 안 되는 데스크탑입니다. 여기선 손가락 사인이 안 되므로 → "타이핑 사인" 병행이 필수. (원하면 화면에 QR을 띄워 폰으로 사인하게 하는 방법도 있음)

결론

방문·원격 어느 쪽이든 커버하려면 → 화면 사인 + 타이핑 사인 둘 다 지원하면 끝. 링크/QR 하나면 병원이 태블릿·폰·데스크탑 어디서든 처리합니다.

7사인을 얼마나 확실하게 받을까 — 본인 확인

QR로 사인을 받으면 "어느 병원 동의서인지"는 자동으로 맞춰지지만, "누가 사인했는지"까지 확인하려면 한 단계가 더 필요합니다. 얼마나 강하게 확인할지 정하면 됩니다.

강도방법확인 수준비용
기본대표자명 입력 + 접속기록(시간·IP)본인 신고무료
중간담당자 번호로 문자 링크 전송 → 그 폰으로만 사인번호 소유 확인문자비(소액)
강함휴대폰 본인인증 / 인증 사인신원 확인유료(외부)

추천

대부분 기본(이름+접속기록)으로 충분합니다. 원격으로 링크를 보낼 땐 "문자 링크(중간)"로 한 단계 올리면 안전합니다. (QR은 그 자리에서 스캔 → 전화번호 불필요 / 문자 링크는 담당자 번호로 전송)

8앱(설치형)은 어떻게?

앱스토어 없이 브라우저에서 "홈 화면에 추가"하면 앱처럼 쓰는 방식(PWA)입니다. 아이폰·안드로이드 공용, 설치·업데이트가 간편하고, 별도 심사·재설치가 없습니다. 개발은 반나절~1일. (같은 방식을 세종시 앱에서 이미 검증)
