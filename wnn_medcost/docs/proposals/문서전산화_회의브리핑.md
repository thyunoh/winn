# WinCheck 컨설팅 문서 전산화 — 회의용 정리 (쉬운 말)

> **한마디로**: 종이·팩스로 주고받던 **의뢰서·견적서·홍보물**을, 병원이 **화면에서 직접 작성·동의·제출**하고 위너넷은 **한 곳에서 관리**하도록 프로그램(WinCheck) 안으로 옮깁니다.

## 1. 무엇을 바꾸나 — 문서 5가지

| 문서 | 지금 | → | 바뀌면 | 순위 |
|---|---|---|---|---|
| 의뢰서 + 동의서 | 인쇄→작성→팩스 회신 | → | 화면에서 입력 + **동의서 사인** | **1순위** |
| 견적서 | 병원마다 수기 작성 | → | 병원 고르고 금액만 → **PDF 자동 발행** | **1순위** |
| 홍보물·회사소개 | 바뀔 때마다 재제작 | → | 화면에서 **문구·숫자 즉시 수정** | 2순위 |
| 프로그램 사용법 | 별도 안내문 | → | 앱 안 **도움말·첫화면 안내** | 3순위 |
| 연간 일정 안내 | 안내문 발송 | → | **공지 + 월별 일정표** | 3순위 |

→ 핵심은 **1순위(의뢰서·견적서)** — 실제 수기 업무(접수·견적)를 없애는 부분. 나머지는 순차 확장.

## 2. 얼마나 걸리나

| 구분 | 기간 | 내용 |
|---|---|---|
| 데모 골격 | **2~3일** | 회의에서 보여줄 뼈대 |
| 핵심 기능 | **1~1.5주** | 의뢰서 + 견적서 + 앱화 |
| 전체 완성 | **1.5~2.5주** | 5개 문서 + 앱 전부 |

## 3. 비용은?

- ✅ **서버·프로그램 추가비용 없음** — 이미 쓰는 무료 오픈소스로 제작. 새 장비·서비스 구매 불필요.
- ⚠️ **딱 하나 — PDF 만드는 도구**: 견적서 PDF 도구(iText)가 **상용 판매 시 라이선스 비용**이 생길 수 있음 → **무료 도구(OpenPDF)로 바꾸면 비용 0** (바꾸는 부담도 적음).
- 문자 알림·인증 사인은 **원할 때만** 유료.

## 5. 동의서 사인 — 두 방식 쉬운 비교

| | 간단 사인 (추천) | 인증 사인 |
|---|---|---|
| 방법 | 화면에 손가락·마우스로 사인 | 휴대폰 인증·공동인증서 |
| 본인 확인 | 이름 입력 + 접속기록 | **있음(강력)** |
| 법적 효력 | 동의 증거(시간·IP) | **강함** |
| 비용 | **무료** | 유료(외부 서비스 계약) |
| 준비 | **바로 가능** | 외부 연동·계약 필요 |
| 언제 | 대부분 컨설팅 동의엔 충분 | 분쟁·감사 리스크 클 때 |

→ **인증 사인이 필요하면** 휴대폰 인증(카카오·PASS) 또는 전자계약 서비스(모두싸인 등)를 **유료 연동**. 자체 구축은 과함. 대부분은 **간단 사인 + 접속기록으로 충분**.

## 6. 동의서를 어떻게 받나 — 방문 vs 원격

> 진짜 갈림길은 기기(폰·태블릿)가 아니라 **방문이냐 원격이냐**.

| 방식 | 서명 상황 | 서명 방법 |
|---|---|---|
| **방문** | 위너넷이 **태블릿 지참** → 현장에서 대표가 손가락 사인 | 화면 사인(깔끔) |
| **원격** | 병원에 **링크·QR 전달** → 병원이 자기 기기로 사인 | 폰·태블릿=화면 사인 / **데스크탑=타이핑 사인** |

- **관건 = 원격 + 데스크탑 병원**: 위너넷은 원격 위주, 병원 접수 담당자는 대부분 **터치 안 되는 데스크탑** → 손가락 사인 불가 → **"타이핑 사인" 병행 필수**. (원하면 화면 QR을 폰으로 스캔해 사인하는 방법도)
- **결론**: 방문·원격 어느 쪽이든 커버하려면 → **화면 사인 + 타이핑 사인 둘 다** 지원. 링크/QR 하나면 태블릿·폰·데스크탑 **어디서든** 처리.

### 참고 — QR 서명이란
데스크탑에서 의뢰서를 입력하고, 서명 단계에서 **화면에 QR이 뜨면 폰으로 스캔 → 폰에서 손가락 사인 → 저장**. 데스크탑↔폰은 직접 연결이 아니라 **서버가 '번호표(토큰)'로 이어줌**(같은 와이파이 불필요). 그 자리에서 스캔하면 **전화번호도 불필요**.

## 7. 사인을 얼마나 확실하게 받을까 — 본인 확인

> QR로 받으면 **"어느 병원 동의서인지"는 자동으로 맞춰지지만, "누가 사인했는지"까지 확인**하려면 한 단계 더 필요합니다.

| 강도 | 방법 | 확인 수준 | 비용 |
|---|---|---|---|
| **기본** | 대표자명 입력 + 접속기록(시간·IP) | 본인 신고 | 무료 |
| **중간** | 담당자 번호로 **문자 링크** 전송 → 그 폰으로만 사인 | 번호 소유 확인 | 문자비(소액) |
| **강함** | 휴대폰 본인인증 / 인증 사인 | 신원 확인 | 유료(외부) |

→ **추천**: 대부분 **기본(이름+접속기록)으로 충분**. 원격으로 링크를 보낼 땐 **"문자 링크(중간)"** 로 한 단계 올리면 안전. (QR=그 자리 스캔·전화번호 불필요 / 문자 링크=담당자 번호로 전송)

## 8. 앱(설치형)은 어떻게?

**앱스토어 없이** 브라우저에서 "홈 화면에 추가"하면 앱처럼 쓰는 방식(PWA). **아이폰·안드로이드 공용**, 설치·업데이트 간편, 심사·재설치 없음. 개발 **반나절~1일**. (같은 방식을 세종시 앱에서 이미 검증)
