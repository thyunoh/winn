전산화의 핵심은 1·2번 의뢰서·견적서입니다 — 실제 수기 업무(접수·견적)를 없애고 앱 데이터와 연계됩니다. 3번 홍보물은 편집형으로, 4·5번은 게시 형태로 전환합니다.

1.2 · 전자 동의 [서식2] 원격접속·DB접근 / [서식3] 개인정보 수집·이용

  • 약관 표시 — 두 동의서 전문을 화면에 노출(스크롤), 병원이 읽고 확인
  • 필수 동의 체크 — 각 동의서 필수 체크박스(둘 다 동의해야 제출 가능)
  • 전자서명 — 요양기관명·대표자명 입력 + 서명(캔버스 서명 또는 성명 타이핑/날인 이미지 업로드)
  • 증빙 기록 — 동의일시·접속 IP·브라우저 정보를 함께 저장해 동의 이력을 남김

1.4 · 관리자 조회 화면 위너넷

  • 접수 목록 — 병원명·요양기관기호·접수일·상태(접수/검토/완료) 필터 및 검색
  • 상세 보기 — 의뢰서 전체 항목 + 담당자 + 동의 이력·전자서명 확인
  • 상태 관리 — 접수 → 검토 → 완료 진행 상태 변경
  • 내보내기 — 엑셀 다운로드, 필요 시 [서식1] PDF 재출력

1.5 · 민감정보 처리 원칙 (필수)

대상: 프로그램 PW, 심평원 인증서 암호 — 유출 시 병원 시스템 접근이 가능한 고위험 정보입니다.

  • 저장 최소화 — 가능하면 저장하지 않고, 원격지원 시점에만 별도 안전채널로 수집·즉시 파기
  • 부득이 저장 시 암호화 — AES-256 양방향 암호화, 복호화 키는 앱과 분리 보관
  • 화면 마스킹 — 관리자 화면에서도 기본 ●●● 표시, 권한자만 일시 조회
  • 접근 통제 · 감사로그 — 지정 담당자만 접근, 조회/복호화 이력 기록
  • 전송 보안 — 전 구간 HTTPS, 보관기간·파기 정책 명시

컨설팅 견적서 온라인 발행

지금은 병원마다 견적서를 수기로 작성합니다(예: 여주병원 ₩4,400,000 VAT 포함). 표준 서식·서비스 범위를 템플릿화해, 관리자가 병원을 고르고 금액만 입력하면 견적 PDF가 즉시 발행됩니다. 견적서는 “계약서 갈음” 역할을 합니다.

홍보물 · 회사소개 콘텐츠 관리

전단(“프로그램 하나로 적정성 준비 끝!”, 위기 7억 손실·기회 3억 질지원금, 3대 기능, 연락처·QR)과 회사소개서를 이미지·인쇄로 관리하면 수치·문구가 바뀔 때마다 재제작해야 합니다. 이를 편집형 페이지(간이 CMS)로 전환해 관리자가 화면에서 직접 수정합니다.

3.1 · 편집 대상 블록

  • 헤드라인·슬로건 — “프로그램 하나로 적정성 준비 끝!”, WINCHECK
  • 위기·기회 수치 — 하위 5% 손실 리스크(약 7억), 상위 20% 질지원금(약 3억) 등 금액·비율 즉시 수정
  • 3대 기능 설명 — 실시간 총점 예측 / 지표 대상자 관리 / 병원 자체 분석
  • 무료제공·이벤트 안내 — 2026년 경영분석 프로그램 추가제공 등 시즌 문구
  • 연락처·QR·검색어 — 전화 02-6953-2452, 네이버 ‘위너넷컨설팅’, QR
  • 회사소개 — 사업분야(교육·컨설팅·전문프로그램), 주요 교육과정

3.2 · 처리(수정) 방식

  • 홍보/소개 페이지 — 병원 대상 소개 랜딩, 로그인 화면 배너로 노출
  • 관리자 편집 — 블록별 문구·수치·이미지·노출여부를 화면에서 수정 → 코드 변경·재인쇄 없이 즉시 반영
  • 수정 이력 — 변경 일시·수정자 기록

4

프로그램 안내 → 앱 도움말

WinCheck+ 사용법 안내문(로그인·자료업로드·대시보드·시뮬레이션)을 앱 안의 도움말로 옮겨, 사용자가 화면에서 바로 확인하게 합니다.

  • 도움말 / 사용가이드 페이지 — 안내문 2~5p 내용을 단계별 화면으로 구성 (로그인 → 자료업로드(SAM·환자평가표·입원현황) → 예상점수 확인 → 시뮬레이션 → 1:1문의)
  • 최초 로그인 온보딩 — 첫 접속 시 3~4단계 간단 안내 팝업
  • 화면 내 도움 아이콘 — 업로드·대시보드 등 주요 화면에 ‘?’ 툴팁 연결

난이도 하 — 대부분 HTML 게시. 기존 FAQ/1:1문의 구조를 재사용할 수 있습니다.

5

연간 진행 안내문 → 공지 페이지

교육 일정·프로그램·보고서 발송 일정을 담은 연간 안내문을 앱 공지로 게시하고, 병원별 일정표로 보여줍니다.

  • 공지사항 페이지 — 연간 진행 안내문(교육·특강·방문교육·보고서 일정)을 HTML 공지로 게시
  • 연간 일정표 — 월별(1~12월) 진행 항목을 표로 시각화 (온라인교육·집체특강·방문교육·보고서 발송)
  • 온라인 교육 링크 — 교육센터(winner797.kr) 안내·계정 규칙(요양기관기호) 연결

난이도 하 — 정보성 게시. 매 주기 안내문만 갱신하면 됩니다.
